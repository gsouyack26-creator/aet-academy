# AET Course - Automation Engineering Technology

**19 Modules | Self-paced | Interactive HTML | Free tools | Progress saved locally**

**Audience:** Industrial maintenance technicians, RME techs, aspiring automation engineers
**Prerequisites:** High-school algebra, basic tool familiarity

## Modules

### Module 0: Introduction to Automation Engineering Technology
- Define AET and its role in modern manufacturing/logistics
- Identify the 10 core technical domains of automation
- Describe how automated systems integrate sensors, controllers, actuators, and networks
- Map AET career paths and credentials (AAS, BSET, certs)
- **Lab:** Explore the Automation Pyramid (Paper/whiteboard)
- **Quiz:** 3 questions

### Module 1: Electrical Fundamentals & Motor-Control Wiring
- Apply Ohm's law and Kirchhoff's laws to DC circuits
- Calculate RMS voltage/current in single/three-phase AC
- Read industrial motor-control schematics (ladder diagrams)
- Wire a 3-wire start/stop motor control circuit
- **Lab:** Start/Stop Circuit (PLCfiddle or pen/paper)
- **Quiz:** 3 questions

### Module 2: PLC Fundamentals
- Explain the PLC scan cycle
- Identify I/O types (discrete/analog, sinking/sourcing)
- Write basic ladder logic: XIC, XIO, OTE, OTL, OTU
- Describe IEC 61131-3 languages (LD, FBD, ST, SFC)
- **Lab:** Build Start/Stop in Ladder (OpenPLC Editor (free))
- **Quiz:** 3 questions

### Module 3: PLC Programming II - Timers, Counters & Data
- Program TON, TOF, RTO timers
- Program CTU/CTD counters
- Use comparison and math instructions
- Organize programs with subroutines/tasks
- **Lab:** Traffic Light Sequencer (OpenPLC or CODESYS (free))
- **Quiz:** 3 questions

### Module 4: Sensors & Instrumentation
- Select sensor types for various measurement needs
- Wire NPN/PNP sensors to PLC inputs
- Interpret 4-20 mA signals and scale to engineering units
- Configure encoder feedback
- **Lab:** Sensor Wiring & Scaling (Pen/paper + calculator)
- **Quiz:** 3 questions

### Module 5: Motors, VFDs & Drives
- Read motor nameplates (HP, FLA, RPM, SF, insulation, frame)
- Explain VFD operation (rectifier-DC bus-inverter, V/Hz)
- Configure essential VFD parameters
- Diagnose common VFD faults (OC, OV, UV, OL, GF, OH)
- **Lab:** VFD Parameter Worksheet (Any VFD manual PDF (free from vendor))
- **Quiz:** 3 questions

### Module 6: Fluid Power - Pneumatics & Hydraulics
- Read ISO 1219 fluid-power symbols
- Design pneumatic circuits with DCVs and cylinders
- Apply Pascal's law for hydraulic force/speed
- Troubleshoot fluid-power faults
- **Lab:** Design Pick-and-Place Circuit (Pen/paper (ISO 1219 schematic))
- **Quiz:** 3 questions

### Module 7: HMI / SCADA Systems
- Design effective HMI screens (ISA-101 principles)
- Configure tags linking HMI to PLC data
- Implement alarm management (ISA-18.2)
- Set up trending/data logging
- **Lab:** Build an HMI Screen (FUXA (free, open-source) or pen/paper)
- **Quiz:** 3 questions

### Module 8: Industrial Networks & Fieldbus
- Compare device/control/enterprise network levels
- Configure Ethernet/IP communication
- Explain OPC-UA and IIoT role
- Troubleshoot network issues
- **Lab:** IP Address Planning (Pen/paper + optional Wireshark)
- **Quiz:** 3 questions

### Module 9: Robotics & Motion Control
- Identify robot types and applications
- Explain coordinate frames and motion types
- Program basic pick-and-place operations
- Apply robot safety standards (ISO 10218, R15.06)
- **Lab:** Pick-and-Place Flowchart (Pen/paper (flowchart))
- **Quiz:** 3 questions

### Module 10: Process Control & PID
- Distinguish open-loop from closed-loop control
- Explain P, I, D terms and effects
- Tune a PID loop systematically
- Read P&ID diagrams (ISA-5.1)
- **Lab:** PID Simulation (Free online PID simulator)
- **Quiz:** 3 questions

### Module 11: Machine Safety & Functional Safety
- Conduct basic risk assessment (severity x probability x avoidance)
- Select safeguarding methods based on risk
- Interpret ISO 13849-1 PL and Categories
- Apply OSHA LOTO and NFPA 70E
- **Lab:** Risk Assessment Exercise (Pen/paper scenario)
- **Quiz:** 3 questions

### Module 12: Capstone - System Integration & Career Paths
- Integrate all AET domains into a complete cell design
- Develop a commissioning checklist
- Map personal skill gaps to learning goals
- Identify certification/degree pathways
- **Lab:** Design Complete Automated Cell (Paper/whiteboard)
- **Quiz:** 3 questions

### Module 13: Advanced PLC - Structured Text, AOIs & Fault Handling
- Write Structured Text with IF/CASE/FOR/WHILE
- Create reusable Add-On Instructions (AOIs) and UDTs
- Implement fault handling (major/minor faults, recovery)
- Design state machines using CASE or SFC
- **Lab:** Build a Motor Control AOI (OpenPLC or CODESYS (free))
- **Quiz:** 3 questions

### Module 14: IIoT & Industry 4.0
- Define IIoT, Industry 4.0, Smart Factory
- Identify edge/fog/cloud architecture layers
- Explain MQTT protocol for industrial data
- Assess OT cybersecurity risks (IEC 62443)
- **Lab:** MQTT Pub/Sub Demo (Mosquitto (free) + MQTT Explorer (free))
- **Quiz:** 3 questions

### Module 15: Electrical Troubleshooting & Test Equipment
- Use DMM safely for V/I/R/continuity
- Interpret megohmmeter insulation tests
- Apply systematic troubleshooting (half-split, signal trace)
- Use oscilloscopes for drive/signal analysis
- **Lab:** Troubleshooting Scenarios (Pen/paper scenarios)
- **Quiz:** 3 questions

### Module 16: Preventive & Predictive Maintenance (Reliability)
- Differentiate reactive/preventive/predictive/prescriptive maintenance
- Calculate MTBF, MTTR, availability
- Interpret vibration analysis basics (FFT, bearing frequencies)
- Design a PM program for automated equipment
- **Lab:** PM Program Design (Spreadsheet or pen/paper)
- **Quiz:** 3 questions

### Module 17: Control Panel Design & Build
- Read/create panel layouts per NFPA 79 / IEC 61439 / UL 508A
- Select and size breakers, contactors, wire, enclosures
- Apply wiring practices: separation, labeling, ferrules
- Perform point-to-point verification and commissioning
- **Lab:** Panel Layout Exercise (Graph paper or diagrams.net (free))
- **Quiz:** 3 questions

### Module 18: Career Acceleration - Portfolio, Interviews & Certifications
- Build a technical portfolio demonstrating AET competency
- Prepare for controls/automation interviews (technical + behavioral)
- Create a certification roadmap by ROI
- Develop a 12-month professional development plan
- **Lab:** Personal Development Sprint (Spreadsheet + this course)
- **Quiz:** 3 questions
