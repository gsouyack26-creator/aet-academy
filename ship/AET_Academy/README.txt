AET ACADEMY - Automation Engineering Technology
Offline training bundle  |  v13.48  |  built 2026-07-22

WHAT'S INSIDE
  index.html                 Double-click this to start (opens the Academy).
  AET_Academy.html           The full learning platform (19 modules, quizzes,
                             flashcards, sims, tracks, certificate, multi-user).
  ElectronWrangler_dist.html Interactive electrical-panel / circuit trainer.
  Quick_Start.pdf            One-page printable guide - print it and keep it
                             at the workstation.

HOW TO USE
  1. Keep all files in the SAME folder - the two apps link to each other.
  2. Double-click index.html (or AET_Academy.html) in any modern browser.
  3. No internet, no install, nothing to configure. Everything runs locally.
     Your progress is saved in the browser (localStorage) per user account.

WORKING TOGETHER
  Many Academy modules have an "Open live panel" button that launches the
  Electron Wrangler trainer at a real ACY1 panel, and a "Try a fault challenge"
  button that opens the panel pre-broken so you can trace the fault. From the
  panel, a "Back to AET Academy" button returns you to the module.

NOTES
  - Chrome, Edge, Firefox all work. Open from a local drive (file://).
  - To reset progress: Profile -> Reset, or clear the browser's site data.
  - Multi-user: create accounts on the sign-in screen; Leads see a Team Roster.
